# Sound Files

Thêm các file âm thanh sau vào thư mục này:

- `shake.mp3` - Âm thanh khi lắc điện thoại
- `bowl-open.mp3` - Âm thanh khi mở bát
- `new-round.mp3` - Âm thanh bắt đầu ván mới
- `new-bet.mp3` - Âm thanh khi có lệnh cược mới
